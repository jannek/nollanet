# Nolla.net

Nollanet is a community website for board riders.

## Environment vars

Not distributed via Github. 

## Create development environment for Windows
```bash
$ . start-win.sh
```

## Create development environment for Mac
```bash
$ . start-mac.sh
```

## Unit tests
```bash
$ python -m unittest -v tests/unit/tests.py
```

## Functional tests
```bash
$ pytest
```
